## ✨ Overview

**Quiz Gaming** is a fully responsive, zero-backend-required trivia platform built with nothing but **pure HTML, CSS & JavaScript**. Players race the clock across three tracks — **HTML, CSS, and JavaScript** — and unlock a downloadable **professional certificate** once they score 80%+ in every category. Scores sync live across every device via an optional free **Firebase Realtime Database** connection.

No frameworks. No build tools. No npm install. Just open `index.html` and play.

<br/>
